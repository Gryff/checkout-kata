# Rules

- Input is a collection of characters eg ABCD
- Output is a price eg 115

Items have the following prices:

- A -> 50
- B -> 30
- C -> 20
- D -> 15

The following *discounts* are also available

- 3 x A -> 120
- 2 x B -> 45

Discounts apply regardless of the order of items in the basket, and apply as many times as they can, eg ABACAAAA -> 290
